# algo out now

![algo cover art](/images/releases/algo-cover.jpg)

I'm so happy with this release! With every new track I feel like it's more and more "me", and this one was the most fun to work on of all of my tracks so far, mostly because I felt more confident with Ableton than I have done previously.

Thanks to [Enigmada](https://lnk.bio/enigmada) and [Circul & Eas](https://circulandeas.com/) for listening feedback, and [Loïc Valenti](https://linktr.ee/loicvalenti) for mastering.
